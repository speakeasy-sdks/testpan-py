# GetAppTelemetriesProtectionStatus

When true, the API will return only protected pods


## Values

| Name              | Value             |
| ----------------- | ----------------- |
| `FULL`            | FULL              |
| `DEPLOYMENT_ONLY` | DEPLOYMENT_ONLY   |
| `CONNECTION_ONLY` | CONNECTION_ONLY   |
| `DISABLED`        | DISABLED          |
| `ALL`             | ALL               |
